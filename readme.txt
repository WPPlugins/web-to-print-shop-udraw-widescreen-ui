uDraw Designer Widescreen UI by Racad Tech

== Description ==
The uDraw Designer Widescreen UI plugin is an extension plugin of [Web To Print Shop: uDraw](https://wordpress.org/plugins/uDraw/ "Web To Print Shop: uDraw"). This plugin allows the user to customize and use a different Designer skin of their own design, than the standard ones bundled in the main plugin. A PDF file is also included for those interested in making their own custom UI. For an online version, please visit https://racadtech.atlassian.net/wiki/display/UDUI/uDraw+Designer+UI. We will update as we add more features to the uDraw Designer.  The uDRAW HTML 5 canvas designer and product personalizer can be obtained via Web to Print Solutions, WebtoPrintShop.com, uDRAW.net and OnlineSignStudio.com

== Installation ==

**Requires**
Plugin: [Web To Print Shop: uDraw](https://wordpress.org/plugins/uDraw/ "Web To Print Shop: uDraw")

1. Unzip and upload contents of the plugin to your /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set this UI as your default UI by selecting it in W2P:uDraw > Settings > Designer UI.
